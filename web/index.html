为什么 / Why

一年前用 cc 干活的 IC 是稀有动物, 现在不是了. 但招聘平台还没赶上 — 他们筛 "会 React", "会 Python", 没有字段筛 "用 agent 把活干漂亮". 这个空白主流平台不会补, 因为补了破坏他们关键词 SEO 模型.

所以另起一条. 协议优先 means: agent-cv.json 和 agent-jobs.json 任何公司都能在自己的 GitHub repo 挂, hireIC 仓库只是众多聚合器之一. 跟 RSS / sitemap.xml / schema.org 一个思路, 不跟 LinkedIn 一个思路.

候选人 cc-fluency 不靠自报, 靠 evidence_url — 公开 PR / commits 里的 Co-authored-by Claude 痕迹 / cc session transcript / 推荐人附的具体 PR. 你说你用得好, 拿东西证明.
